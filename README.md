# touch-bistro-challenge-app

## To Install the Application

1) Install the node packages with:
   ```
   npm install
   ```
2) Run the metro serve with
   ```
   npm start
   ```
3) To run the app in iOS
   ```
   npm run ios
   ```
4) To run the app in Android
   ```
   npm run android
   ```

## Testing is performed with jest

To run tests

```
npm run test
```
